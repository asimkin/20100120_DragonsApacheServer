Dragons Apache Server Lite 1.1
Данная версия является облегченной модификацией сборки Apache HTTP Server и содержит минимально необходимый набор компонентов для разработки и тестирования web-сайтов.
В данную сборку входят следующие компоненты:
# Apache 2.2.14
# PHP 5.2.12
# MySQL 5.1
# Server Monitor 1.1.0.1
Внимание! Данную сборку крайне не рекомендуется использовать как сервер 24/7 для сайтов, форумов и т.п.
Авторы сборки не обязаны прибегать к вам по первому зову и оказывать поддержку в режиме 24/7. Кроме того, авторы могут отказаться от проекта и его поддержки, в случае если их бесконечный альтруизм иссякнет.
Внимание! Если у вас включен Skype необходимо его выключить перед запуском сервера или в настройках Skype на вкладке "Соединение" убрать галку "Использовать порты 80 и 443 в качестве входящих альтернативных".
